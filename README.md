# Estudos Web Workflow

Projeto desenvolvido com o curso Git a Web Developer Job: Mastering the Modern Workflow, para estudo de tecnologias Git, Node.js, NPM, JavaScript, ES6, webpack, Gulp e BEM.

### Pré-requisitos

```
Node.js
```

### Instalação

```
npm install
```


### Executar

```
npm run dev
```
